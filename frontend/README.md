# ForgeOS Frontend

Next.js 15 (App Router) + TypeScript + Tailwind CSS v4 + shadcn/ui.

```bash
npm install
cp .env.local.example .env.local   # edit if your backend is not on localhost:8000
npm run dev                        # http://localhost:3000
```

Full documentation, architecture, security notes, and deployment guide: see the
[root README](../README.md).
